# Solar System Mod (SSM)
![logo](https://github.com/Eissentam/SSM-Solar-System-Mod/blob/main/sprites-override/ui/logo.png)
> 一个为Mindustry添加太阳系内容的模组。

**请注意：本模组已于 2023-2-25 停止更新，进入维护状态。详情见下文。**

## 📜 免责声明与重要说明

- **‼️‼️注意‼️‼️**本模组会严重破坏游戏平衡！。
- **当前的游戏兼容性**：目前模组可以在mindustry154中运行（部分方块因为游戏的更新，导致无法使用，比如**大型发射台等**）
>理论上，只要Anuken不再像136那次更新一样，应该是可以正常运行的。
- **贴图来源说明**：本模组在开发初期借鉴了 **饱和火力** 的部分贴图和Json文件。若觉不妥，请联系我们进行删除或修改。
- **模组性质**：开发初期的贴图由一人完成，风格较为简单。如觉影响视觉体验，建议游玩更高品质的模组。
- **项目状态**：本模组主要功能因为技术原因无法继续，目前暂无主动开发计划。最后一次更新为 **2023年2月25日**。

## 🛠️ 下载与安装

1. 确保你的游戏版本与本模组兼容。
2. 下载最新的模组文件（`zip` 或通过模组管理器安装）。
3. 在游戏的模组界面加载模组。
4. 重启游戏并确认模组加载成功。
